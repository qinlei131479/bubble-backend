## 项目概述（智能如泡，聚变未来）

**AgetBubbles** 是一个面向下一代人机交互的智能体原生平台，将大语言模型、RAG 检索增强生成与自主智能体技术深度融合，
以“泡泡”为核心理念，构建轻量化、可组合、自进化的 AI 应用生态。平台旨在让每个想法都能如泡泡般轻盈萌发，
自由碰撞，融合成更强大的智能形态，真正实现 AI 能力的民主化与场景化落地。

**AgetBubbles** 是一个微服务开发平台开发的前端项目。它利用了现代 Web 技术，包括 Vue.js 3、Element Plus 和
Vite，提供了一个健壮且高效的开发环境。

后端仓库：https://github.com/qinlei131479/BubbleCloud.git

## 功能特性

- **Vue 3**: 利用最新版本的 Vue.js 实现现代化的响应式体验。
- **Element Plus**: 集成了 Element Plus，提供丰富的 UI 组件。
- **Vite**: 使用 Vite 进行快速构建和模块热替换。
- **TypeScript**: 支持 TypeScript，提升代码质量和可维护性。
- **Tailwind CSS**: 使用 Tailwind CSS 进行样式设计。

## ✨ 核心特性（未来规划）
### 智能体即泡泡（Agent-as-Bubble）架构

- **轻量封装**：每个智能体被封装为独立“泡泡”，具备自治的感知-决策-执行循环
- **动态组合**：泡泡可通过标准化接口自由连接，形成动态工作流（泡泡簇）
- **热插拔部署**：支持运行时智能体的动态加载、卸载与升级，零停机更新

### 三层智能融合引擎

- **大模型中枢**：集成多模态 LLM 统一调度层，支持混合推理与成本优化
- **RAG 增强矩阵**：
智能文档解析与向量化流水线
多源检索融合（向量+关键词+图检索）
基于会话的上下文动态增强
- **智能体运行时**：内置推理框架、工具调用库与记忆管理系统

### 泡泡协作网络

- **智能路由**：任务自动匹配最优智能体或智能体组合
- **竞合机制**：支持多智能体协同求解与结果投票融合
- **能力共享**：知识、工具、技能可在泡泡间安全共享

### 开发者友好设计

- **低代码编排**：可视化泡泡工作流编辑器，拖拽式智能体组装
- **模板市场**：预置行业智能体模板（客服、研发、分析等）
- **一键部署**：从原型到生产环境的全链路支持

### 企业级管控

- **权限隔离**：项目级、智能体级、API 级多层权限控制
- **审计追踪**：全链路可观测性，对话溯源与行为日志
- **合规支持**：数据本地化、敏感信息过滤、合规性检查

## 快速开始

### 先决条件

- **Node.js**: 版本 18.0.0。
- **npm**: 版本 8.0.0 或更高。

### 使用文档

npm install  
npm run dev

## 浏览器支持

- 现代浏览器的最后两个版本。
- 不支持 IE 11 及更低版本。
